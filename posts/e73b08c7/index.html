10月29号，Notepad++在官网发布了新版本v7.8.1，看到之后，PP希望大家立刻！马上！卸载Notepad++！

废话不多说，看看这个程序猿脑子怎么了!

官网内容大概的意思就是我们如何的差，抹黑我们，并且希望通过Notepad的力量来施压等等
但是，nodepad++开发者完全在放屁

维吾尔族自始至终就是中国的一部分，并且从1999年开始到现在，西部开发战略也没有停下过。不仅给西部带去了资源，还有发达的技术，有无数志愿者投身西部开发，甚至付出生命。

经过数十年的努力，西部各地区得到了高速发展，人民的生活也得到大大的改善。

对于这样的言论，Github的网友们反应更加强烈。看到网友们的评论，直呼过瘾，这种人就该被骂。

Notepad++的作者竟然说出这样的言论，小编第一反应就是某些外媒搞鬼，但百度之后才发现，Notepad++的作者是台湾出生，现法国籍。并且在2008年奥运会的时候，在Notepad++发布“抵制 2008 北京奥运会”。
当时就引起许多人弃用Notepad++。

我不知道身处台湾的侯今吾有没有来过大陆，如果来过就知道大陆现在的发展，如果来过就知道在国家的带动下维吾尔族的现状。

看侯今吾对大陆的排斥，估计没来过吧。

那就给他免费科普一下历史。

维吾尔族居住的新疆地区从公元前60年就已成为中国的一部分！公元前138年，汉武帝就派张骞出使西域，开始了新疆地区与内地的联系。公元前60年，匈奴的日逐王被迫降汉，西汉在吐鲁番设立西域都护府，管理天山南北各地，从此西域统一于中央政府。清政府统一新疆后，于1762年10月设立伊犁将军府，在新疆实行以伊犁将军为首的军事统治。1882年沙俄被迫归还中国伊犁地区以后，清朝政府于1884年在新疆建省。

自此之后，中国经历百年变革，新疆地区依旧是中国不可分割的一部分。

如果你还是看不懂，那就看看这个！

中华人民共和国地图！新疆是中国的一部分！台湾是中国的一部分！香港也是！
中华5000年历史对你要求太高，你就把中华人民共和国地图学一下吧！

PP不会再用Notepad++，又不是不可替代。诸多网友也表示如此。

在这里给大家推荐3款可替代的免费软件。
Sublime Text
官网: https://www.sublimetext.com/

无限使用

具有漂亮的用户界面和强大的功能,例如代码缩略图,Python的插件,代码段等.还可自定义键绑定,菜单和工具栏.Sublime Text 的主要功能包括：拼写检查,书签,完整的 Python API , Goto 功能,即时项目切换,多选择,多窗口等等.Sublime Text 是一个跨平台的编辑器,同时支持Windows、Linux、Mac OS X等操作系统.

Atom
官网: https://atom.io/

免费

Atom 代码编辑器支持 Windows、Mac、Linux 三大桌面平台,完全免费,并且已经在 GitHub 上开放了全部的源代码.在经过一段长时间的迭代开发和不断改进后,Atom 正式版在性能和稳定性方面都有着显著的改善. 与大多数其他编辑器相比,Atom的语言支持已经算是覆盖非常全面了.另外,它的代码补全功能（也叫Snippets） 也非常好用,你只需输入几个字符即可展开成各种常用代码,可以极大提高编程效率.

VS Code

官网: https://www.visualstudio.com/en-us/products/code-vs.aspx

免费

vscode同样是一个跨平台的文本编辑器,内置了对JavaScript, TypeScript and Node.js,C++, C#, Python, PHP等语言的强大支持。

从Notepad++2008年抵制奥运开始到今天，无数的中国人用实际行动告诉Notepad++一个事实：

什么都是可以替代的，而我的祖国——中国，是不能替代的！

香港在秦汉时期就是中国的一部分；新疆在公元前60年就是中国的一部分；台湾更是在夏商时期就是中国的一部分。

这些东西是刻在骨子里的，是改国籍也改不掉的！

PP深知技术人员不参与政治话题，但看到搞技术的人抹黑我的祖国，那就不行！因为我身为技术人员之前，我是名中国人！以前是，现在是，以后也是。
你确实很优秀，用的人很多，但是这不代表我只能用你！

从今天起，我不再用Notepad++，为信仰献出绵薄之力。
